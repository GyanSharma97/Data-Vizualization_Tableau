## Dataset Description

The project uses the **Global Superstore dataset**, which contains transactional retail data from multiple international markets.

### Main Tables

**Orders**
Contains order-level information including:

- Order ID
- Order Date and Ship Date
- Customer details
- Product information
- Sales and profit
- Quantity and discounts
- Shipping cost
- Order priority
- Region and market

**Returns**
Contains information about returned orders.

**People**
Maps regional managers to specific regions.

- Customer Segment
- Order Priority

## Dashboard Insights

The Tableau visualizations help answer questions such as:

- Which regions generate the highest sales?
- Which product categories contribute the most profit?
- How do discounts affect profitability?
- Which markets have the highest shipping costs?
- What customer segments drive the most revenue?

The dashboard makes it easier to explore patterns in the data and identify areas where performance can be improved.
